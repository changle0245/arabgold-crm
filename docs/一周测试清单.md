## 测试约定

- 每个 `[ ]` 是一个独立可勾选的测试点
- **测试账号准备**：
  - admin 账号（如 `admin@arabgold.test`，密码 `test12345` 已重置）
  - 至少 2 个 member 账号（用于权限隔离测试）
- **数据准备**：
  - 至少 100 个客户，分布在多个国家、阶段、业务员
  - 至少 50 个报价、20 个成交、10 个样品
  - 至少 20 条提醒（含逾期 / 今天 / 未来）
- **失败处理**：每条测试 fail 时记录到本文档底部「问题清单」
- **浏览器**：主测 Chrome；P1-P2 测试在 Edge/Firefox/Safari 重复关键流程
- **本测试清单不包含**：单元测试 / Playwright 自动化（前者代码层面、后者已在 docs/上线前待办清单.md 标记 fail）
